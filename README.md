# rise-for-archlinux-hyprland
🌅 Rise Setup for Hyprland 

В файле info расписаны все дериктории для утилит и некотоыре полезные команды и комбинации клавиш.
Дополнительно прекреплен файл hyprland.conf это файл отвечающий за настройку системы.

Так же прекрепленны 2 фотографии:
- **gay.png** - используется в hyprlock
- **1854_1.png** - используется в fastfetch

Минималистичная конфигурация для Arch Linux + Hyprland с темой **Rise**. Включает готовые настройки для:

- **Fastfetch** — информативный аскетичный вывод
- **Waybar** — прозрачная панель с soft‑градиентом
- **Hyprlock** — элегантный экран блокировки
- **Kitty** — терминал с теплой цветовой схемой

📦 Установка

```
# Клонируем репозиторий
git clone https://github.com/homyak1208/rise-for-archlinux-hyprland.git
cd rise-hyprland
```

## 📂 Копирование файлов

> ⚠️ Рекомендуется сделать бэкапы существующих конфигов.

```
# Fastfetch
cp -r config/fastfetch/* ~/.config/fastfetch/

# Waybar
cp -r config/waybar/* ~/.config/waybar/

# Hyprlock
cp -r config/hyprlock.conf ~/.config/hypr/

# Kitty
cp -r config/kitty/* ~/.config/kitty/
```

## 🚀 Применение

Перезапустите соответствующие сервисы:

```
killall waybar && waybar &
killall hyprlock
fastfetch --gen-config
```
Так же советую использование:

- **Caelestia** — кастомизированный рисовый инструмент, заменяет лаунчер, waybar, подстраивает тему под фон, имеет смену фона(при установке настройка цветовой схемы в kitty не требуется, тема подстраивается сама)
- **Starship** — минималистичный prompt с иконками и временем выполнения команд
- **Fish** — умная оболочка с подсветкой синтаксиса и автодополнением

Так же могу посоветовать удобные/веселые утилиты:

- **Audacious** — лёгкий аудиоплеер с ретро-эстетикой и поддержкой плейлистов
- **nwg-look** — GTK-темизация под единый стиль окружения Hyprland
- **Cava** — консольный аудиовизуализатор с плавными баровыми спектрами
- **LavaT** — динамический эффект лавы в терминале для атмосферного рисового настроения
- **tty-clock** — аналоговые часы в терминале в стиле старого доброго CLI
- **cbonsai** — растущее ASCII-дерево бонсай, успокаивающий скринсейвер или просто украшение

Готово — наслаждайтесь темой **Rise** 🌄
